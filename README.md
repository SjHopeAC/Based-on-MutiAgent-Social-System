# 基于多智能体的社会系统模拟

本仓库包含一个多模块实验项目，主要涉及垃圾评论检测与社区平台演示。

## 项目结构

- **helloworld** – Java Springboot项目
- **spam_comment_detection_system** – 垃圾评论识别系统（Python），包含算法模型和核心架构
- **vue_demo** – 前端演示界面（Vue.js）
- **.vscode** – VSCode 编辑器配置文件

## 快速开始

- Java 项目：进入 `helloworld` 目录，使用 IDE 或 `javac` 运行
- Python 项目：进入 `spam_comment_detection_system`，安装依赖后运行
- Vue 项目：进入 `vue_demo`，执行 `npm install` 和 `npm run serve`

详细用法请参考各子目录下的说明。
